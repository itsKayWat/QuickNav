Appleseed-QuickNav

Purpose

Appleseed-QuickNav is designed to streamline navigation within the Appleseed Repairs POS system by providing customizable keyboard shortcuts for quick access to both local and web-based resources.

Key Features

  • Custom keyboard shortcuts for frequent pages
  • Local file access support
  • GitHub sync for settings backup
  • Dark mode interface
  • Import/Export configuration

Installation Requirements

  • Google Chrome Browser
  • Local HTML files for POS system
  • Optional: GitHub account for settings sync

Installation Steps

  1. Open Chrome Extensions page (chrome://extensions/)
  2. Enable Developer Mode
  3. Load unpacked extension from the Appleseed-QuickNav folder
  4. Configure shortcuts in extension options

Use Case Example

A service technician needs to quickly switch between check-in forms, diagnostics, and parts ordering pages throughout the day. Instead of navigating through folders, they can press simple keyboard combinations like "4" for Check-In or "6" for Order Parts.
